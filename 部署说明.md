# 部署建议

## 推荐路线
### 路线1：GitHub Pages
1. 新建一个 GitHub 仓库
2. 上传 `web_exp1_online` 中全部文件
3. 在仓库 Settings -> Pages 中启用部署
4. 获得线上链接后发给被试

### 路线2：Netlify / Vercel
1. 上传整个 `web_exp1_online` 文件夹
2. 选择静态网站部署
3. 部署完成后获得公网链接

## 数据回传建议
如果需要自动集中收数，建议你后续加一个后端：
- 腾讯云 / 阿里云轻量服务器
- Firebase Functions / Supabase Edge Functions
- 一个简单的 PHP / Node.js 接口

接口只需要接收浏览器 POST 的 JSON，并保存到数据库或文件即可。

## 浏览器限制建议
为了保证数据质量，建议在招募说明中写清楚：
- 仅限电脑端
- 全屏完成
- 不要中途切换页面
- 建议使用 Chrome 或 Edge
