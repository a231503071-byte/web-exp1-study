# 自动在线收数配置说明

## 方案 A：你自己有服务器
1. 选择 `server_examples/node_express` 或 `server_examples/php`
2. 把接口部署到你的服务器
3. 在前端 `config.js` 里填入：
```js
webhookUrl: "https://你的域名/submit"
```
4. 重新部署前端网页

## 方案 B：先本地部署测试
- 前端本地静态服务器
- 后端本地 Node.js 接口
- `webhookUrl` 填：
```js
webhookUrl: "http://localhost:3000/submit"
```

## 方案 C：先不做自动上传
如果你暂时没有服务器，可以先保持：
```js
webhookUrl: ""
```
这样被试完成后仍会自动下载 JSON/CSV 到本地。

## 数据安全建议
- 正式收数时不要只依赖浏览器本地下载
- 建议同时：自动上传 + 本地下载双备份
- 服务器目录应设置访问权限，避免直接公开被试结果文件
