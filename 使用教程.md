# 使用教程

## 1. 本地测试
```bash
python -m http.server 8000
```
浏览器打开：
`http://localhost:8000`

## 2. 情绪组分配
默认随机分到 PH / PL / NH / NL。
也可在启动页选择手动组别。

## 3. 自动收数
### Node.js
进入 `server_examples/node_express`：
```bash
npm install
npm start
```
然后把 `config.js` 改为：
```js
webhookUrl: "http://localhost:3000/submit"
```

### PHP
把 `server_examples/php/save_exp1.php` 上传到服务器，并在 `config.js` 填入对应 URL。

## 4. 结果输出
实验结束后浏览器会自动下载：
- JSON 原始结果
- summary_cn.csv
- trials_cn.csv
- recall_cn.csv


## 当前版本的编号与分组规则
- 被试编号：仅接受 `W001`、`W002`、`W003` 这类线上样本编号
- 分组方式：固定为程序随机分组
- 组别：自动随机分配到 `PH / PL / NH / NL`
- 数据中会分别记录：
  - 被试编号
  - 样本来源（线上）
  - 组别
