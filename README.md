## 运行方式
不要直接双击 `index.html`，请先启动本地静态服务器：

```bash
python -m http.server 8000
```

然后打开：
`http://localhost:8000`

## 自动收数
- 不配置 `webhookUrl`：只本地下载结果
- 配置 `webhookUrl`：自动上传 + 本地下载双备份

## 当前网页版流程
1. 被试信息填写
2. 前测评分：效价、唤醒
3. 情绪图片观看
4. 后测评分：效价、唤醒、趋近、回避、行动冲动
5. 中性词记忆
6. 倒数任务（电脑输入）
7. 自由回忆
8. 努力/难度/疲劳评分
9. 自动导出 JSON 和中文 CSV
