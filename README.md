# 实验1在线版
## 已纳入的材料
- 中性词表：实验1_中性词语_ANCW筛选_40词.xlsx
- 词语图片：40 张
- 指导语图片：62 张
- 情绪图片：
  - PH：20 张
  - PL：20 张
  - NH：20 张
  - NL：20 张

## 运行方式
不要直接双击 `index.html`，请先启动本地静态服务器：

```bash
python -m http.server 8000
```

然后打开：
`http://localhost:8000`

## 自动收数
- 不配置 `webhookUrl`：只本地下载结果
- 配置 `webhookUrl`：自动上传 + 本地下载双备份

## 当前网页版流程
1. 被试信息填写
2. 前测评分：效价、唤醒
3. 情绪图片观看
4. 后测评分：效价、唤醒、趋近、回避、行动冲动
5. 中性词记忆
6. 倒数任务（电脑输入）
7. 自由回忆
8. 努力/难度/疲劳评分
9. 自动导出 JSON 和中文 CSV

## 说明
这是一套根据完整实验材料重建的网页行为实验程序，适合实验1的线上扩样。



## 当前版本的编号与分组规则
- 被试编号：仅接受 `W001`、`W002`、`W003` 这类线上样本编号
- 分组方式：固定为程序随机分组
- 组别：自动随机分配到 `PH / PL / NH / NL`
- 数据中会分别记录：
  - 被试编号
  - 样本来源（线上）
  - 组别
