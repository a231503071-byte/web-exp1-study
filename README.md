## 运行方式
不要直接双击 `index.html`，请先启动本地静态服务器：

```bash
python -m http.server 8000
```

然后打开：
`http://localhost:8000`

## 自动收数
- 不配置 `webhookUrl`：只本地下载结果
- 配置 `webhookUrl`：自动上传 + 本地下载双备份

## 当前网页版流程
1. 被试信息填写
2. 前测评分：效价、唤醒
3. 情绪图片观看
4. 后测评分：效价、唤醒、趋近、回避、行动冲动
5. 中性词记忆
6. 倒数任务（电脑输入）
7. 自由回忆
8. 努力/难度/疲劳评分
9. 自动导出 JSON 和中文 CSV

## 自动收数版说明
本版本已经附带 PHP 接收接口示例：
- server_examples/php/save_exp1.php

你需要：
1. 把 save_exp1.php 上传到支持 PHP 的服务器
2. 把 config.js 中的 webhookUrl 改成真实地址
3. 再把前端上传到 GitHub Pages

正式收数建议：
- 自动上传 + 本地下载 双备份


## 自动编号版说明
本版本新增：
- server_examples/php/assign_code.php

用途：
- 让系统自动从 W001 - W128 中分配一个未使用编号
- 避免被试都手动写成 W001

注意：
- 只有在配置了服务器编号接口 `codeServiceUrl` 后，才能真正保证不重复
- 如果不配置服务器接口，网页只能本地随机编号，不能保证绝对唯一
